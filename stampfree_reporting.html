New audit template

Follow these steps to replicate this report for a new scrape or sector

Step 1

Run the scraper

Update the company list in trustpilot_scraper.py and run. Output lands as a timestamped JSON in the same folder. Make sure filterStars: ["1","2","3"] is set as strings.

Step 2

Add a new tab to this file

Duplicate the audit tab HTML block and paste it below. Rename the tab label and pane ID. Click "Refresh with new scrape" and drop in the new JSON — metrics and bar chart update automatically.

Step 3

Update the editorial sections

The quotes and opportunities table need a manual review each run — the data surfaces the signal but the commentary needs your judgement. Update the pane subtitle and methodology note to reflect the new date range and company set.
